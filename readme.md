# 💻 C++ Error Analyzer

## 📌 Description

This is a command-line based C++ Error Analyzer developed using Object-Oriented Programming concepts.
It detects common errors in C++ code and provides simple explanations and suggested fixes.

---

## 🚀 Features

* 🔍 Analyzes C++ code line-by-line for common programming errors
* ⚠️ Detects missing semicolons in variable declarations and statements
* 🧩 Identifies unmatched or improperly balanced brackets using stack logic
* 📌 Tracks variable declarations and flags undeclared variable usage
* 💡 Provides clear explanations for each detected error
* 🛠️ Suggests possible fixes to help users correct mistakes
* 📊 Displays error type along with line number for easy debugging
* 🧱 Built using Object-Oriented Programming principles (inheritance & polymorphism)


---

## 🧠 OOP Concepts Used

* Encapsulation
* Inheritance
* Polymorphism
* Abstraction

---

## 🛠️ Technologies Used

* C++
* Standard Template Library (STL)

---

## ▶️ How to Run

### 1. Compile

g++ analyzer.cpp -o analyzer

### 2. Run

./analyzer

---

## 📂 Input Method

* Enter C++ code line by line
* Type `END` to finish input

---

## 🧪 Sample Input

int main() {
int a = 5
cout << a;
}
END

---

## ⚠️ Limitations

* Not a full compiler
* Uses pattern-based error detection
