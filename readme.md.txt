# 💻 C++ Error Analyzer

## 📌 Description

This is a command-line based C++ Error Analyzer developed using Object-Oriented Programming concepts.
It detects common errors in C++ code and provides simple explanations and suggested fixes.

---

## 🚀 Features

* Detects missing semicolons
* Detects unmatched brackets
* Detects undeclared variables
* Beginner-friendly error explanations

---

## 🧠 OOP Concepts Used

* Encapsulation
* Inheritance
* Polymorphism
* Abstraction

---

## 🛠️ Technologies Used

* C++
* Standard Template Library (STL)

---

## ▶️ How to Run

### 1. Compile

g++ analyzer.cpp -o analyzer

### 2. Run

./analyzer

---

## 📂 Input Method

* Enter C++ code line by line
* Type `END` to finish input

---

## 🧪 Sample Input

int main() {
int a = 5
cout << a;
}
END

---

## ⚠️ Limitations

* Not a full compiler
* Uses pattern-based error detection

---

## 🎯 Learning Outcome

* Better understanding of OOP concepts
* Implementation of polymorphism
* Basics of static code analysis
